Base de Datos
	Conjunto organizado de informacion estructurada que se almacena y gestiona de manera sistematica

	-Se compone por tablas de filas y columnas que representan datos/registros
	-Existen relacionales(PostgreSQL)/No relacionales(MONGODB) y NoSQL(No tabla)
	-Facilita la gestion de grandes cantidades de datos

DBMS->Data Base Management System
	-Datos: Informacion o hechos que se pueden medir,observar o recopilar. Se almacenan y uutilizan para tomar decisiones informadas y obtener informacion util 
	-Base de datos: Coleccion ORGANIZADA de datos que se almacenan en una computadota y se pueden acceder y actualizar facilmente

DBMS RELACIONALES: MYSQL POSTGRESQL,SQLITE,
DBMS NO RELACIONALES(NO SQL): MONGODB, FIREBASE

SQL->Structured Query Language
	Lenguaje de consultas utilizado para administrar y consultar bases de datos relacionales
	SQL permite a los usuarios crear,modificar y actualizadatos almacenados en una base de datos de manera eficiente
